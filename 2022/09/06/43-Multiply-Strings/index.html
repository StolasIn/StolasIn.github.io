Multiply Strings

📖 description

給定兩個由數字組成的字串，回傳他們相乘後的結果。

ex.
    Input: num1 = "123", num2 = "456"
    Output: "56088"

🧠 solution

使用字串模擬乘法，要注意處理修改的位數以及處理進位。

⏳ time complexity

處理兩個字串的每個位數相乘的結果，時間複雜度 O(mn) ( m 為 num1 的長度, n 為 num2 的長度)
處理字串反轉，時間複雜度為 O(2m+2n)
總時間複雜度 O(mn)
